## Testing suite
This project demonstrates Unit Testing and Interaction Testing in a React application using:

Jest (test runner)
React Testing Library (RTL) (component testing)
userEvent (real user interaction simulation)

The goal is to ensure that UI components not only render correctly but also behave properly when users interact with them.

---
## Components

| Component | Description                      |
| --------- | -------------------------------- |
| Button    | Displays text passed via props   |
| Card      | Displays a title                 |
| Input     | Controlled input field           |
| Counter   | Increments value on button click |

---
## Running Tests
```
npm test
```
Press a to run all tests
